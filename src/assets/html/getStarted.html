Télécharger

Pour le fonctionnent optimal de MaterialTech, télécharger et installer jQuery.

MaterialTech est disponible dans une version complète  ou par fichier. Sélectionnez votre version selon vos préférences et votre expertise.

Si vous voulez utiliser l’intégralité de la librairie télécharger le fichier MaterialTech et rajoutez-le dans votre code HTML.

<script src="MaterialTech.min.js"></script>

Si vous voulez un effet en particulier, télécharger le fichier JavaScript homonyme de l’effet souhaite.

<script src="ripple.min.js"></script>

Vous pouvez aussi  utiliser les fichiers en ligne.

<span><!-- MaterialTech minifieé --></span> <script src="https://josilopera.com/materialtech/materialtech.min.js"></script> <span><!-- Effet Ripple minifieé --></span> <script src="https://josilopera.com/materialtech/ripple.min.js"></script> <span><!-- Effet Modal minifieé --></span> <script src="https://josilopera.com/materialtech/modal.min.js"></script> <span><!-- Effet Social minifieé --></span> <script src="https://josilopera.com/materialtech/social.min.js"></script>

Initialiser la librairie depuis de votre code JavaScript, modifiez les paramètres ou laisser vide si valeurs par défaut.

<span><!-- Valeurs par défaut --></span> $(my_target).ripple({}); <span><!-- Personnalisé --></span> $(my_target).mt_modal({ "speed1": "1000", "speed2": "600", "animation1": "easeOutSine", "animation3": "easeOutSine", "layout": "left", "closeColor": "#4e2d17", "finalBackground": "#7f7eec"});

Vous pouvez modifier les paramètres depuis l’application web à l’aide du panel de control en choisissant l’effet désiré dans le menu de navigation. Rappelez vous récupérer le code dans l’onglet  « Afficher votre code ».
